# セキュリティチェックリスト

実装の最終確認時に、Claude Code が必ずチェックする項目。

## 機密情報

- [ ] API キー・トークン・パスワードがコードに直書きされていないか。
- [ ] `.env` / `.env.local` が `.gitignore` に含まれ、追跡対象になっていないか。
- [ ] `console.log` で機密情報（トークン・キー・個人情報）を出力していないか。

## 危険な動的実行

- [ ] `eval(` を使っていないか。
- [ ] `new Function(` を使っていないか。
- [ ] その他、信頼できない文字列を実行する経路を作っていないか。

## 外部通信

- [ ] 外部 API・外部サービスへの新規通信に明確な理由があるか。
- [ ] 通信先 URL がハードコードされていないか（環境変数で管理されているか）。

## 依存関係

- [ ] 新規依存関係に明確な理由があるか。
- [ ] メンテナンス状態が極端に悪いパッケージを採用していないか。

## ユーザーデータ

- [ ] ユーザーデータ・保存データを破壊的に変更していないか。
- [ ] マイグレーションが必要な場合、明示指示を受けたか。
